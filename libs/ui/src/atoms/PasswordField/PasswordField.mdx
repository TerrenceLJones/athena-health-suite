# PasswordField

Wraps `TextField` with a built-in show/hide visibility toggle — a `type="password"`/`type="text"`
switch plus an eye icon button rendered as the field's suffix action.

A shared primitive so every Provider password entry point (sign-in today, and any password-reset
flow later) uses one accessible toggle instead of each screen re-implementing the same show/hide
boilerplate (a `useState`, an icon, and a button).

## Behavior

- Defaults to masked (`type="password"`).
- The toggle button's accessible name flips between "Show password" and "Hide password", so
  screen readers announce the action rather than just an icon change.
- All other `TextField` props (`label`, `state`, `error`, `help`, ...) pass through unchanged;
  only `type` and `suffix` are owned internally and not exposed.
